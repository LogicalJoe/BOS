fs_GetSectorPtr

syscall Adress 020114

InputsWhat it doesOutputsDestroysNotes
void *fs_GetSectorPtr(void *fd, int sector); read a given sector of a file descriptor hl = pointer to sector. hl = -1 if failed. this does not guarantee a contiguous memory space, as files can be fragmented in FAT32.
